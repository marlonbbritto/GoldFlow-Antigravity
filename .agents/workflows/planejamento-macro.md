# Workflow: Planejamento Macro (/planejamento-macro)

Fase estratégica de ideação e desenho do roadmap do produto.

## Fluxo de Trabalho
1. **Brainstorming (@po)**: Explora a ideia do usuário e define os objetivos do produto usando a skill de brainstorming.
2. **Recomendação de Stack (@architect)**: Analisa a complexidade e **sugere** o conteúdo para o \STACK.md\ (sem alterá-lo diretamente).
3. **Roadmap (@agile)**: Lista as User Stories necessárias para o MVP/Projeto.
4. **Persistência**: Salva o plano em \docs/blueprint/blueprint-[nome].md\.

**Nota**: O usuário deve revisar a stack sugerida e atualizar o \STACK.md\ manualmente se concordar.

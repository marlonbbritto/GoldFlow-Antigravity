# 🚀 GoldFlow Antigravity: Squad de Desenvolvimento Autônomo

![Arquitetura GoldFlow](docs/img/diagrama-golden-flow.png)

O GoldFlow é um framework de engenharia de software orientado a agentes de IA, projetado para garantir que a automação siga rigorosos padrões de qualidade e governança técnica.

## 🏗️ Ciclo de Vida do Projeto (The Golden Flow)
A governança técnica é centralizada no \STACK.md\, tornando o fluxo dinâmico e adaptável a qualquer tecnologia.

0. **Estratégia (\/planejamento-macro\):** Fase de descoberta e ideação. O squad utiliza a skill de brainstorming para desenhar o roadmap macro e salvar em \docs/blueprint/\.
1. **Definição (\STACK.md\):** Onde são ditadas as regras tecnológicas e os padrões arquiteturais do jogo.
2. **Planejamento (\/criar-us\):** O squad detalha os requisitos INVEST e a solução técnica em \docs/user_stories/\.
3. **Implementação (\/desenvolver-us\):** Execução técnica via ciclo TDD, orquestrada pelo agilista e auditada por segurança.
4. **Validação:** Homologação rigorosa com testes E2E reais via Browser Subagent.

## 🧠 O Squad (Personas)
Conforme definido no \AGENTS.md\, cada agente possui responsabilidades e restrições estritas:

* **@po**: Focado em requisitos de negócio, modelo INVEST e critérios de aceite.
* **@architect**: Guardião da arquitetura (\STACK.md\), Clean Code e princípios SOLID.
* **@qa_eng**: Responsável pela execução de testes E2E utilizando o **Browser Subagent** nativo.
* **@agile**: Orquestrador do fluxo, responsável por decompor planos em subtarefas atômicas.
* **@backend_dev / @frontend_dev**: Especialistas em implementação técnica e testes automatizados.
* **@sec**: Auditor de segurança estrutural e sigilo de arquivos internos.
* **@devops**: Gerente de ambiente, terminal e controle de versão.

## 🛡️ Protocolo Anti-Loop
Para máxima eficiência, o squad opera sob o protocolo:
- **Prioridade I/O**: Comandos de salvar arquivos têm precedência sobre refinamentos de pensamento.
- **Veto ao Kaizen**: Melhorias contínuas são pausadas após a aprovação do rascunho de salvamento.
